# Instruções para Ícone

Você precisa adicionar um ícone `.ico` para o sistema aparecer corretamente na bandeja do Windows.

## Opções:

### 1️⃣ Usar um ícone online:
Baixe um ícone de: https://icon-icons.com/pt/download/158390/ICO/256/
Salve como: `icons/crc-icon.ico`

### 2️⃣ Converter uma imagem:
Use: https://convertico.com/
- Faça upload de uma imagem PNG/JPG
- Baixe o arquivo .ICO
- Salve em `icons/crc-icon.ico`

### 3️⃣ Usar emoji como ícone (temporário):
O sistema vai funcionar sem ícone customizado, usando o ícone padrão do Electron.
